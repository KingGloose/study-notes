# 卡片模板

## 知识卡片模板

用于学习阶段，记录对话中提炼的核心洞察。

```markdown
# 知识卡片 #[编号]

**主题**：[一句话描述]

**核心观点**：
[用户的独特见解或主张]

**支撑论据**：
1. [论据1]
2. [论据2]
3. [论据3]

**用户金句**：
> "[用户原话中最有价值的表达]"

**延伸问题**：
- [可以进一步探讨的问题]

**来源对话**：第X-Y轮

---
生成时间：[时间戳]
```

## 输出卡片模板

用于结构阶段，规划文章每个部分的内容。

```markdown
# 输出卡片 #[编号] - [节点标题]

**在文章中的位置**：[开篇/中段/结尾]

**核心论点**：
[一句话概括这部分要传达的信息]

**展开要点**：
1. [要点1 - 简述]
2. [要点2 - 简述]
3. [要点3 - 简述]

**素材来源**：
- 知识卡片 #X：[引用内容]
- 知识卡片 #Y：[引用内容]

**写作指引**：
- 语气：[正式/轻松/故事性...]
- 修辞：[类比/数据/案例...]

**预估字数**：XXX 字

**状态**：[ ] 待写作 / [ ] 已完成 / [ ] 已润色

---
```

## 项目信息模板

用于追踪整个写作项目的状态。

```markdown
# 项目：[项目名称]

## 基本信息
- **创建时间**：
- **目标读者**：
- **预期字数**：
- **截止日期**：

## 当前状态
- **阶段**：学习 / 结构 / 写作 / 成稿
- **进度**：X%

## 知识卡片清单
| 编号 | 主题 | 状态 |
|------|------|------|
| #1   |      |      |

## 输出卡片清单
| 编号 | 节点 | 字数 | 状态 |
|------|------|------|------|
| #1   |      |      |      |

## 备注
[任何需要记录的信息]
```
